# MouseMover project

Микроприложение под Windows, которое двигает мышкой по гругу на 1 пиксель в секунду. Вправо, вниз, влево, вверх и по кругу.

## Requirements

- Имеет два состояния:
	- On - включен, мышка двигается по кругу
	- Off - выключен, никаких событий не посылается
- Запускается всегда свёрнутым в system tray
- Иконка в system tray отображает текущее состояние (мышь активна / неактивна)
- Дабл клик по иконке в system tray переключает состояние на другое
- Правый клик мыши по иконке system tray открывает контекстное меню:
	- "Turn on" (если выключено) или "Turn off" (если включено) - переключает состояние
	- "Autostart" - чекбокс; нажатие переключает состояние
		- Включен - автостарт в реестре для данного приложения и данного пользователя прописан
		- Выключен - записи в реестре нет
	- Разделитель
	- "Exit" - завершает приложение
